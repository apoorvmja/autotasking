# Glossary

- Admin: User who manages destinations, interns, prompts, and YouTube uploads.
- Intern: User who signs in to complete daily tasks.
- Destination: A posting target (Reddit, Facebook, YouTube, Other).
- Prompt: Reddit-only template with `{{group}}` and `{{url}}` placeholders.
- Daily task: One task per destination generated once per IST day.
- Status: Per-day completion flag stored in `*_task_status` tables.
- YouTube upload: A video file + metadata stored in Supabase storage and `youtube_videos`.
- IST day: Day boundary defined by UTC+05:30.
